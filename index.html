Simple inventory & account demo — quick register and login flows.

Open Register View on GitHub

Note: the app backend uses PHP/MySQL (`db.php`). Netlify serves the static landing page only — PHP must be hosted elsewhere.

Quick start

  • To run locally: `php -S localhost:8000` from project root.
  • Ensure `db.php` points to a reachable MySQL instance.
  • Netlify will serve this static landing page; PHP will not execute on Netlify.
